# Cypress Automation Project

This project contains a collection of automated web tests using **Cypress**, demonstrating end-to-end testing of common web functionalities.

The project covers a variety of common scenarios, including:

- **Mouse Hover**: Verify hover effects on elements.
- **Checkbox Handling**: Select, unselect, and validate checkbox states.
- **Clear Text**: Automate clearing text from input fields.
- **Dynamic Dropdown Menu**: Select items from dynamic dropdowns.
- **Login Functionality**: Automate login validation with valid and invalid credentials.
- **Radio Buttons**: Select and verify radio button options.
- **User Registration**: Automate new user registration.
- **Re-registration / Duplicate User Handling**: Test registration with existing accounts.
- **Scroll Tests**: Validate scroll behavior on pages.
- **File Upload**: Automate file upload functionality.
